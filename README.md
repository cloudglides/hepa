# DIY Air Purifier

Simple project to build your own air purifier using a HEPA filter(phiphilps filter used for refrence and a 150 mm fan. Print the case, slam the parts together join them with heatsink and screws and it should be tougher than most of the filters out there



## BOM.CSV

- **Filter:** Philips FY0900 compatible HEPA  (any hepa filter but i got only a phiphilps air purifier)
- **Fan:** 150 mm HEWA exhaust fan (any fan as long as it fits)
